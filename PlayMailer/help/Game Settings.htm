Game Settings

From here you can change settings specific to each game, such as a game's folder path, or the command to run a game. Select a game from the drop down list and change settings as needed. Most games will have the following settings:
  • Game Folder Path -- Enter the path to the game's install folder. Use the [...] button on the right to browse to the folder.
  • Game Run Command -- Enter a custom run command to launch the game. You should not need to change this setting unless you have a specific reasonn. To change this settings, first untick Use default run command then change the run command. Note: By default many games are configured to run the emulator DOSBox, however it is not possible to change the run command to use a different emulator as PlayMailer only supports DOSBox currently.
  • Use default run command -- if this is ticked then PlayMailer will automatically determine the game's run command based on the Game Folder Path.
  • Run in full-screen -- If ticked, PlayMailer will launch the game in full-screen, otherwise it will be launched in a normal window.
