Notify Settings

These settings affect the way notifcation windows popup when displaying status updates and chat messages.
  • Chat and Status Windows Top-Most
    • Always -- Chat and status windows will popup in front of all other windows regardless of whether you are currently using PlayMailer. 
    • Only when game window active -- Chat and status windows will popup top-most only when you are currently in a game window. Otherwise they will popup behind the currently active window and flash the window's icon in the taskbar.
  • Mute Chat
    • Mute all chat messages -- If ticked, you will not receive any chat messages. The messages will be deleted.
    • Mute session-wide chat message -- If ticked, you will not receive session-wide shouts, however you will still receive private chat messages. Session-wide messages will be deleted.
  • Disable Sounds
    • Disable 'Chat Received' sound -- If ticked, a sound will not be played when the chat window pops up. 
    • Disable 'Your Turn' sound -- If ticked, a sound will not be played when the 'Your Turn' status window pops ups.
