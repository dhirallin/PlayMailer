Using a Session

Once you have created a new session, an email will be sent to all players inviting them to join the session. Players will then be asked to enter a Player Name and any other required player settings. Players can also modify the game's settings at this time or edit the player list, e.g. to add additional players..
1) Wait until it is your turn. A window will popup informing you when it is your turn.
2) Click the Load Now button and wait for the game to load fully.
Note: PlayMailer will automatically setup the game, configure all players and load your turn for you.
Important: Do not use your mouse or keyboard while PlayMailer is loading the game. 
3) Have your first turn. But do not press 'End Turn' at the end of your turn.
4) When you are finished your turn press the following Hotkey Ctrl + Shift + S from within the game window to save the game and send it to the next player.
5) Wait until your next turn. You may close the game or switch windows if you desire.
Note: If you ever accidently press End Turn in your game, you will have to re-load the game from the PlayMailer window and redo your turn.

Session Actions

Various actions can be performed on a Session from the main PlayMailer window by clicking on the small buttons next to a session in the session list. The following actions are available:
  • Load -- Launches the game and starts your turn.  Same as clicking on the Load Now button in the Your Turn window. You can use this option if you originally selected Load Later.
  • Re-Send -- Re-sends a turn request to the player following you in the turn-order. This allows the player following you to re-take their turn. This action can be used in the following circumstances:
    • The player following you in the turn-order has accidently deleted or corrupted their session and needs to have the session information re-sent to them.
    • The game needs to be rolled back to an earlier turn. E.g. you can roll-back the game to the player immediately following you in the turn-order, regardless of whose turn it is.
    • Note:  This action does not export the current save file from your game. Therefore if you wish to re-take your turn then you need to use the in-game hotkey CTRL + SHIFT + S to resend the game afterwards.
  • Players -- Displays the Player List window for a session. From here you can view a session's players, including their emails, names and teams. The player who is currently having their turn is marked with an arrow. From this window you can also send chat messages to indivisual players or the entire session or edit the player list. 
  • Settings -- Displays the Session Settings window. From here you can change the session name, edit the player list, or change any game specific settings. Important:  Game options that are available from this window should not be changed manually within the game itself. Game options are controlled by PlayMailer to ensure that they remain consistent across all players' copies of the game. 
  • Delete -- Deletes the session. Note: This will only delete the session on your computer. Other remote players will receive a status update informing them that you have been removed from the player list.  

In-game Hotkeys

PlayMailer provides several in-game Global Hotkeys for performing actions on the currently active session.  A session becomes the currently active session after pressing the Load or Load Now buttons to load your turn. Alternatively you can set a session as the currently active session by clicking on it in the main PlayMailer window. The currently active session is highlighted in the session list.
The global hot-keys can be used from within the game, even if the game is in full-screen, or from any other window. Exception: the Send Game hotkey, CTRL + SHIFT + S, must be used from within the game window.
To view a list of the available hotkeys, press the View Hotkeys button in the PlayMailer window.

Using Chat

PlayMailer can be used to send chat messages to other players. You can access chat from within your game, or from any other window by using global hotkeys. To send a chat message to all players in a session press CTRL + SHIFT + M and type in ynour message.
Note: The session you wish to send a chat message to must be the currently active session, so you first need to either Load your turn for that session or select the session from the PlayMailer window.
To send a chat message to a single player, or a selection of players - Press Ctrl + Shift + P to bring up the player list for the currently active session. Tick the players you want to send a message to and press Send Message To Selected.
You can also send messages from the PlayMailer window by clicking on the Players button next to a session and then using Send Message To Selected or Send Message To All.

When you receive a chat message from another player, you can send a reply using the following reply options:
  • Reply to Sender -- sends a private reply to the sender of the message.
  • Reply to Sender + Recipients -- Sends a reply to the sender of the message plus all other recipients of the message.
  • Reply to All -- sends a session-wide shout to all players in the session belonging to the sender.
After you have read a chat message, you can use the Next button to cycle through further chat messages, or one of the following Skip options to skip further messages:
  • Skip All Messages -- Deletes the remaining chat messages without viewing them.
  • Skip Session-wide Messages -- Deletes the remaining session-wide shouts, but continues cycling through private messages.
Note: If you find that chat messages keep popping up and bothering you while you are playing a game, you can tick Mute all chat messages or Mute session-wide chat messages from the PlayMailer window.
Alternatively, if you find that chat messages are popping up while you are busy working in another application, you can select Chat and Status Windows Display Top-Most --> Only when game window active, from the Notify Settings page in Settings.
